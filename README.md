# Selenium with Python :robot:

---

Dois scripts que enviam mensagem pelo whatsapp. 

É preciso ter instalado na maquina o Python e um editor de código-fonte (vscode por exemplo).

## Esse repositório cintem:

- **whats-bot-script** envia uma mensagem de texto simples;
- **whats-bot-img** envia uma imagem com legenda.

Antes de tudo, no Terminal execute os comandos 'pip install selenium' e depois 'pip install webdriver_manager'.

[Selenium with Python documentation](https://selenium-python.readthedocs.io/)
